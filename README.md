## Краткое описание

Модуль **awz.dealsproductfield** добавляет новый тип пользовательского поля для Bitrix CRM, позволяющий выбирать сделки и конкретные товары из товарного набора этих сделок.

## Подробное описание

Модуль расширяет функциональность пользовательских полей Bitrix, добавляя тип поля `awz_deals_product`. Это поле предназначено для работы в CRM-системе и позволяет:

- **Выбирать сделки** из CRM с использованием стандартного селектора сущностей (EntitySelector)
- **Выбирать товары** из товарного набора каждой выбранной сделки
- **Просматривать детали** сделки и список её товаров с ценами и количеством
- **Работать с множественными полями** — можно добавить несколько разных сделок с выбранными товарами

### Основные возможности

1. **Интерфейс выбора**: Использует стандартный UI EntitySelector Bitrix для удобного выбора сделок
2. **Автоматическое определение контекста**: При редактировании поля в карточке сделки, эта сделка автоматически добавляется в список выбранных
3. **Хранение данных**: Данные хранятся в формате JSON с информацией о сделках и выбранных товарах
4. **API контроллер**: Предоставляет методы для получения списка товаров сделки, информации о сделке и поиска сделок
5. **Права доступа**: Проверяет права чтения сделок перед отображением данных
